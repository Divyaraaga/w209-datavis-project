Dragon Boat Performance tracking Dashboard

Dragon boat is a speed sport, for which my team practices daily. The data visualized here, which has been collected from ERGO Dragon Boat trainer represents my performance, in terms of number of paddles, in daily practice sessions.
Daily practice has laps of three different durations, 3 min, 5 mins and 8 min that are aimed at increasing paddling speed and stamina. Each lap type has a specific target number of paddles for each individual, e.g 180 paddles is the target for a 3 min interval drill. It is very important for each individual to achieve this target in order to synchronize overall team effort.
This dashboard helps me to check the difference between the target and actual number of paddles I achieved. As the dashboard shows, my performnce in 3 minute drills is consistent, but in the other two drills there is a difference between the target and my actual performance. This difference is shown on top of each bar in the chart.

Interactions

  1. Hover over the bars to see the target, actual and deviation from target.
  2. Click on different intervals from the legend to filter the bars accordingly
